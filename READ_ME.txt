Create an App with initial Login flow, Use Google Sign In to login into the app. On the Home Screen,  use Google Maps and Add a search Bar on top of the Screen. Use Google Places API to search for a Place using Address in the search bar and show a list of possible addresses. Tapping on Google Places result item should Display the “PIN” of the location on the MAP. 

You should be able to mark any place as favourite and display Favourites in another screen. 
You can use a Tab View to display both Home and favourites. Users should be able to unmark a location as their favourite.
